## Authority Model

- The audit document states what is broken now.
- The target architecture document defines what must exist when the work is complete.
- The implementation plan defines build order, file-level work, and acceptance criteria.
- The validation gates document defines what evidence is required before the work is considered done.
- The handoff document is the single agent-facing execution brief.
