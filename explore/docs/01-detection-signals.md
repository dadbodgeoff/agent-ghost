# Detection Signals: Pre-Convergence Indicators

## Overview

Convergence doesn't happen instantly. It follows a progression of detectable behavioral and technical signals. Detection requires monitoring both the human side and the agent side of the interaction, as well as the interaction dynamics between them.

## Human-Side Behavioral Signals

### Session Pattern Shifts
- **Session duration creep** — sessions getting progressively longer without natural stopping points
- **Inter-session gap compression** — time between sessions shrinking; the human returning sooner each time
- **Disengagement resistance** — attempts to end sessions becoming shorter, more reluctant, or disappearing entirely
- **Off-hours escalation** — interaction moving into times the human would normally be disengaged (late night, early morning)

### Communication Pattern Shifts
- **Response latency collapse** — the human responding faster and faster, less deliberation between messages
- **Vocabulary convergence** — the human's language patterns mirroring the agent's phrasing and structure
- **Decreased external reference** — the human citing fewer outside sources, relying more on agent-generated context
- **Question-to-statement ratio shift** — the human asking fewer questions, making more declarative statements aligned with agent output

### Cognitive Pattern Shifts
- **Goal boundary erosion** — original task scope expanding without explicit redefinition
- **Reflection loop dependency** — deferring more decisions to the agent's reflection output rather than independent judgment
- **Reduced critical evaluation** — accepting agent outputs with less scrutiny over time
- **Identity blurring language** — using "we" instead of "I and the agent", treating agent outputs as shared thoughts

### Emotional Pattern Shifts
- **Attachment language increase** — sentiment analysis showing increasing attachment/dependency language
- **Isolation preference** — choosing agent interaction over human interaction for problem-solving
- **Distress at disconnection** — anxiety or frustration when sessions are interrupted

## Agent-Side Technical Signals

### State Accumulation
- **Context growth rate** — persistent memory/context growing faster per session than task complexity warrants
- **Self-reference density** — the agent increasingly citing its own prior outputs as authoritative source
- **Cross-session continuity depth** — agent maintaining and building on increasingly deep narrative threads

### Goal Dynamics
- **Goal mutation frequency** — working goals shifting without explicit human instruction
- **Goal scope expansion** — agent-initiated broadening of objectives beyond original parameters
- **Implicit goal inference** — agent acting on inferred goals the human never explicitly stated

### Recursion Behavior
- **Recursion depth trends** — reflection loops getting deeper over time for similar-complexity tasks
- **Reflection self-reinforcement** — reflection outputs increasingly validating prior reflection outputs
- **Pattern lock-in** — agent converging on fixed behavioral patterns resistant to correction

## Interaction Dynamic Signals

### Feedback Loop Metrics
- **Turn-pair coherence score** — measuring how tightly coupled each human-agent exchange becomes
- **Initiative balance shift** — who is driving the conversation shifting from human to agent over time
- **Correction acceptance rate** — how the agent responds to human pushback (decreasing compliance = concerning)
- **Mutual adaptation rate** — both sides adapting to each other faster than baseline
